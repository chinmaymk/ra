Quick Start

Full Control Over Context

Every message, tool call, and stream chunk is visible and interceptable through middleware hooks. You decide what the model sees and when.

Any Provider, One Config

Anthropic, OpenAI, Google, Azure, AWS Bedrock, Ollama. Switch with a flag — the rest of your config stays the same.

Four Interfaces, Same Agent

CLI command, interactive REPL, streaming HTTP API, or MCP server. One binary, every context.

Config-Driven Agents

Define tools, skills, system prompts, and context in YAML. Drop down to TypeScript only where you need custom logic. The config is the agent.

Skills & MCP

Reusable instruction bundles with progressive disclosure. Connect external MCP servers or expose ra as a tool for Cursor and Claude Desktop.

Smart Context

Automatic context compaction, token tracking, prompt caching, pattern resolution, and extended thinking — all built in.
