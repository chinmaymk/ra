Get Started
Quick Start

Full Control Over Context

Every message, tool call, and stream chunk is yours to inspect, modify, or stop. Middleware hooks run at every step. You decide what the model sees and when.

Any Provider, Any Model

Use any model from Anthropic, OpenAI, Azure OpenAI, Google Gemini, AWS Bedrock, or Ollama. Switch with a flag — the rest of your config stays the same.

Any Interface

Ship as a CLI command, interactive REPL, streaming HTTP API, or MCP server for Cursor and Claude Desktop. One binary, every context.

Skills & MCP

Define behavior through skills and system prompts. Connect external MCP servers for tools. Expose ra itself as an MCP tool. Extend without touching core code.

Layered Config

defaults → file → env → CLI. Commit a baseline, override per-run. The config is the agent — change the config, change the agent.
